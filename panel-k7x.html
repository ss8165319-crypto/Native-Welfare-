ℹ️ How it works

Settings are saved to the settings collection in Firestore (document ID: main).
Your index.html reads these on load via onSnapshot and applies them instantly — no redeploy needed.
Paste this snippet in your index.html script to consume the settings: 

onSnapshot(doc(db, 'settings', 'main'), (snap) => {
  const s = snap.data() || {};
  if (s.title)        document.title = s.title;
  if (s.heroHeading)  document.querySelector('.hero-h1 .line1').textContent = s.heroHeading;
  if (s.heroDesc)     document.querySelector('.hero-sub').textContent = s.heroDesc;
});
